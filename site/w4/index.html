    Week 4 report

    Problem Statements

    Clarify the 5 Ws

    Five Ws * Who Who is using the site * What What is the site offering/doing,what should the main focus be * Where Online * When ASAP, later? * Why Why create/redesign the site

    Feature List

    Scope - what are you going to do/implement for this website redesign

    e.g. - update branding (colors, typography) - redesign home page layout - redesign ____ page layout - redesign menu

    Ideation

    Generate as many possible solutions to the problem as you can Non-judgemental

    Techniques - Brainstorming - Worst Possible Idea - Crazy8s (generate quick sketches (wireframes))

    Information Architecture (IA)

    Organizing, labelling of content - to help users find information

    Comprised of several 'systems': - navigation (what type of menu, how many menus, hierarchy) - hierarchy deep (very few top menu links with a lot of links underneath) shallow (many top menu links with very few links underneath) - labelling (what style of language do we use) - content inventory (listing all of the pages/content that the website) - sitemap (a visual representation of all the content) - wireframes (simple line drawings depicting the layout of menus/pages)
